# CodeAlpha Image Gallery

A responsive image gallery built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features

* Responsive Image Grid
* Hover Effects
* Lightbox Popup
* Next and Previous Navigation
* Modern UI Design
* Mobile-Friendly Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

CodeAlpha_ImageGallery/

├── index.html

├── style.css

├── script.js

├── images/

└── README.md

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in your browser.
4. Click any image to view it in the lightbox.
